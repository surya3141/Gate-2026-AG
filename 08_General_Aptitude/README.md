- Full GA section mock tests
- Analyze mistakes
- Focus on accuracy over speed

### Exam Strategy

**Time Management:**
- **Target:** 22-25 minutes for 15 marks
- **1-mark questions:** 45-60 seconds each
- **2-mark questions:** 90-120 seconds each

**Attempt Strategy:**
1. **Quick scan** (30 sec): Identify easy questions
2. **Solve easy first** (10 min): Secure 8-10 marks
3. **Medium difficulty** (10 min): Attempt carefully
4. **Difficult/time-consuming:** Skip or guess strategically
5. **Review** (2-3 min): Check silly mistakes

**Guessing Strategy:**
- **1-mark:** Negative -1/3 (guess if 50%+ sure)
- **2-mark:** Negative -2/3 (guess only if 60%+ sure)
- Don't leave blank if you can eliminate 2+ options
