## ðŸ’» During Exam (9:30 AM - 12:30 PM)

### **First 5 Minutes (9:30-9:35):**
- [ ] **DON'T rush to start answering!**
- [ ] Read all instructions carefully
- [ ] Check total questions, marks distribution
- [ ] Note negative marking scheme
- [ ] Plan rough time allocation

### **Time Management Strategy:**

**Phase 1: Easy Questions (First 60 minutes)**
- [ ] Quickly scan ALL questions once (5 min)
- [ ] Mark easy questions mentally
- [ ] Start with General Aptitude (22-25 min)
  - Quick, fresh mind advantage
  - Secure 10-12 marks minimum
- [ ] Then core subjects easy questions (30-35 min)
  - Direct formula questions
  - Concepts you're 100% sure of
- [ ] **Target:** 40-45 marks in first hour

**Phase 2: Medium Difficulty (Next 80 minutes)**
- [ ] Attempt medium difficulty questions
- [ ] Multi-step problems
- [ ] Questions requiring 2-3 minutes
- [ ] Use virtual calculator efficiently
- [ ] **Target:** Additional 30-35 marks

**Phase 3: Tough/Time-consuming (Next 40 minutes)**
- [ ] Attempt difficult questions strategically
- [ ] If stuck >3 minutes, mark for review and move
- [ ] Eliminate wrong options, make educated guesses
- [ ] **Target:** 10-15 marks

**Phase 4: Review (Last 20 minutes)**
- [ ] Review all marked questions
- [ ] Check silly calculation errors
- [ ] Revisit skipped questions
- [ ] Final guessing (if confident)
- [ ] **DON'T change answers unless sure!**

### **Virtual Calculator Tips:**
- Basic calculator provided on screen
- Practice using it beforehand
- Use rough sheet for complex calculations
- Double-check decimal points

### **Negative Marking Awareness:**
- 1-mark: -1/3 mark for wrong answer
- 2-mark: -2/3 mark for wrong answer
- **Guess only if >50% confident (1-mark) or >60% (2-mark)**

### **Rough Work:**
- Use provided scribble pad/rough sheets
- Organize calculations section-wise
- Don't waste time making it neat
- Cross out used space to avoid confusion
