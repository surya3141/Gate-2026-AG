## üìã Must-Know Concepts

### HACCP (Hazard Analysis Critical Control Points)
```
CCP = Critical Control Point
- Step where control prevents/eliminates hazard
- Examples: Pasteurization, metal detection, cooling

7 Principles:
1. Hazard analysis
2. Determine CCPs
3. Critical limits
4. Monitoring
5. Corrective actions
6. Verification
7. Records
```

### Proximate Analysis (Standard 5)
```
1. Moisture (105¬∞C oven)
2. Protein (Kjeldahl N √ó 6.38)
3. Fat (Soxhlet)
4. Ash (550¬∞C muffle)
5. Fiber (crude fiber)
6. Carbohydrate (by difference)

"MPFFA-C" = Moisture-Protein-Fat-Fiber-Ash-(Carb)
```

### Sensory Evaluation
```
9-Point Hedonic Scale: PREFERENCE testing
9 = Like extremely
5 = Neither like nor dislike
1 = Dislike extremely

Other methods:
- Triangle: Difference (find odd one)
- Duo-trio: Match to reference
- Ranking: Order preference
```

### Type 1: Acidity Calculation (1 min)

**Given:** V = 5 ml, N = 0.1, W = 10 g

**Formula:** Acidity% = (V √ó N √ó 0.009 / W) √ó 100

**Calculate:**

= (0.0045 / 10) √ó 100
= **0.045%** or **0.45% lactic acid**

### Type 2: Protein from Nitrogen (30 sec)

**Given:** N = 0.53% in milk

**Formula:** Protein% = N% √ó 6.38

**Calculate:** = 0.53 √ó 6.38 = **3.38% protein**

### Type 3: Moisture wb% (30 sec)

**Given:** Wet weight = 50 g, Dry weight = 10 g

**Formula:** Moisture% = [(Wet - Dry) / Wet] √ó 100

**Calculate:** = [(50-10)/50] √ó 100 = (40/50) √ó 100 = **80% wb**
